# Load Skill 지침

새로운 세션이 시작되거나 프로젝트의 전체적인 흐름을 빠르게 파악해야 할 때 이 스킬을 사용합니다. 이 스킬은 불필요한 파일 탐색과 연구를 줄여 컨텍스트 쿼터를 효율적으로 관리하게 해줍니다.

## 실행 절차

1.  **progress.md 읽기**: 루트 디렉토리에 있는 `progress.md` 파일을 읽습니다.
2.  **맥락 분석**: 
    - 가장 최근에 완료된 작업은 무엇인지 확인합니다.
    - 현재 진행 중이거나 다음에 해야 할 작업이 무엇인지 파악합니다.
    - 프로젝트의 전반적인 기술 스택과 아키텍처 결정을 이해합니다.
3.  **결과 요약**: 
    - 사용자에게 현재 파악한 상태를 간결하게 브리핑합니다.
    - 파악한 맥락을 바탕으로 바로 이어서 진행할 수 있는 구체적인 제안을 합니다.

## 주의 사항
- `progress.md`가 없을 경우, 사용자에게 현재 상태를 물어보거나 최소한의 탐색을 통해 새로 생성할 것을 권장합니다.
- 정보를 나열하기보다는 '현재 어디까지 왔고, 무엇을 할 것인지'에 초점을 맞춰 요약하세요.
