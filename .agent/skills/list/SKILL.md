# List Skill 지침

사용자가 'list' 또는 이와 유사한 요청을 할 경우 다음 절차를 따르세요.

## 실행 절차

1.  **스킬 디렉토리 탐색**: `.agent/skills/` 디렉토리 내의 모든 하위 폴더를 확인합니다.
2.  **스킬 정보 추출**:
    - 각 폴더의 `SKILL.md` 파일을 읽습니다.
    - YAML 프런트매터에서 `name`과 `description` 필드를 확인합니다.
3.  **리스트 출력**:
    - 확인된 스킬들을 깔끔한 테이블 또는 리스트 형식으로 사용자에게 보여줍니다.
    - 예시:
        | 스킬 이름 | 설명 |
        | :--- | :--- |
        | commit | 프로젝트 진행 상황 기록 및 커밋/푸시 |
        | list | 사용 가능한 스킬 리스트 출력 |

## 주의 사항
- 각 스킬의 `SKILL.md` 파일이 존재하는지 먼저 확인하세요.
- 사용자에게 친절하고 명확한 설명을 제공하세요.
